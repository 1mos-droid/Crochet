A full-stack e-commerce web application designed for browsing and purchasing handcrafted crochet items. This project demonstrates a modern shopping experience using vanilla frontend technologies backed by a robust Express.js server.

Frontend:
 * HTML5: Semantic structure and layout.
 * CSS3: Custom styling, animations, and responsive design.
 * JavaScript (ES6+): DOM manipulation, cart logic, and API fetching.
Backend:
 * Node.js: Runtime environment.
 * Express.js: RESTful API routing and static file serving.
âœ¨ Features
 * Product Showcase: dynamic gallery of crochet items (amigurumi, clothing, accessories) fetched from the backend.
 * Shopping Cart: Add/remove items, adjust quantities, and view real-time total price calculations.
 * Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.
 * RESTful API: Express backend handles product data requests and serves static assets.

ğŸš€ Getting Started
Follow these instructions to set up the project locally.
Prerequisites
 * Node.js (v14 or higher)
 * npm (Node Package Manager)
Installation
 * Clone the repository
   git clone [https://github.com/1mos-droid/Crochet.git](https://github.com/1mos-droid/Crochet.git)
cd Crochet

 * Install backend dependencies
   npm install

 * Start the server
   npm start
# OR for development with nodemon:
# npm run dev

 * Access the App
   Open your browser and navigate to:
   http://localhost:3000 (or whichever port is defined in your console output).
