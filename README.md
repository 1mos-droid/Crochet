**Installation**

1.    
      
    **Clone the repository:**  
    git clone [https://github.com/1mos-droid/Crochet.git](https://github.com/1mos-droid/Crochet.git)  
    cd Crochet  
      
      
      
2.    
      
    **Install dependencies:**  
    npm install  
      
      
      
3.    
      
    **Start the server:**  
    npm start  
    \# or for development with automatic reloads:  
    npm run dev  
      
      
      
4.    
      
    **Access the Application:** Open your browser and navigate to: http://localhost:3000
